## STRING 
* string is a text or comination of both number and text
* we have to use double quotes while provideing a default value to a string "value"
### SYNTAX
```terraform
variable "Name" {
   type = "string"
   default = "hari krishna"
}
```
### SYNTAX
```terraform
variable "ami_id" {
   type = "string"
   default = "ami-0b4f379183e5706b9"
}
```
